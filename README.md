# Globopersona Frontend Redesign

A modern frontend redesign for the Globopersona email marketing and automation platform built using Next.js, React, TypeScript, and Tailwind CSS.

## Features

* Responsive SaaS dashboard UI
* Campaign management pages
* Contacts management
* Create campaign page
* Dark / Light mode
* Authentication pages
* Analytics charts
* Reusable UI components
* Responsive layouts

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Recharts
* Lucide React

## Installation

Clone the repository:

```bash
git clone https://github.com/srinidhxi/Globopersona-Redesign.git
```

Navigate into the project folder:

```bash
cd Globopersona-Redesign
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
